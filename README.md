# AfyaTech Precision Prompting Challenge

## Overview
This project contains rewritten AI prompts for an SMS-based maternal health assistant designed for rural communities in Kenya and Uganda.

The prompts were improved using:
- AIM Framework (Actor, Intent, Mode)
- MAP Framework (Milieu, Audience, Purpose)
- Chain-of-Thought Reasoning
- Verifier Pattern for safer AI responses

## Assignment Goals
The objective of this assignment is to:
- Improve prompt precision
- Reduce generic AI responses
- Create culturally relevant healthcare guidance
- Design safer emergency triage interactions
