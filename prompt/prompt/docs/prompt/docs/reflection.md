# Reflection

This exercise shows that AI in healthcare is only as good as its context. Generic outputs fail when they ignore real-world constraints like distance, income, and cultural practices.

Precision prompting transforms AI into a decision-support tool rather than just an information source. It also highlights the importance of structure, especially in high-risk scenarios like emergency triage.

Going forward, AI systems must be designed around lived realities. Without that grounding, even accurate information can become ineffective or unsafe.
